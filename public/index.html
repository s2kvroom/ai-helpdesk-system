When to Use Live Agent Support

Live support is best for urgent, sensitive, or more complex issues that need direct troubleshooting from a real person.

Use live help for situations like:

  • Account lockouts that prevent you from signing in
  • Network outages affecting work, school, or home access
  • Software crashes that keep happening after basic troubleshooting
  • Hardware problems like printers, laptops, or monitors not working
  • Urgent work interruptions where downtime matters
  • Permission or access issues involving sensitive systems

Standard AI help is usually enough for:

  • Password reset guidance
  • Common troubleshooting steps
  • Basic software setup questions
  • Simple error message explanations
